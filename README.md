# E-commerce-stud-app

# 🛒 E-commerce Backend in Python (OOP-Based)

A simple, console-based E-commerce backend application built using **Object-Oriented Programming (OOP)** principles in Python. This project simulates core functionalities such as user registration, product management, cart handling, and order processing.

---

## 📌 Features

- 🧑 User registration (in-memory simulation)
- 📦 Product listing and stock management
- 🛒 Shopping cart (add, remove, view items)
- ✅ Order placement with total amount calculation
- 🧠 Follows OOP concepts:
  - Encapsulation
  - Inheritance (if extended)
  - Abstraction
  - Single Responsibility Principle

---

## 🛠️ Tech Stack

- **Language**: Python 3.x
- **Architecture**: Object-Oriented
- **Persistence**: In-memory (no database)
- **Interface**: Console-based (CLI)
